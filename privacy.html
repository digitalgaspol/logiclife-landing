Kebijakan Privasi

Terakhir diperbarui: 27 Februari 2026

Keamanan dan privasi data Anda adalah prioritas utama LogicLife Digital. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan ekosistem aplikasi kami.

1. Informasi yang Kami Kumpulkan

Untuk memberikan pengalaman terbaik dan memproses penarikan saldo Anda, kami mengumpulkan beberapa data berikut:

  • Data Akun: Alamat email, nama pengguna, dan foto profil (jika login via Google).
  • Data Perangkat: ID Perangkat (Device ID) dan versi sistem operasi untuk mencegah pembuatan akun ganda/tuyul.
  • Data Transaksi: Nomor e-Wallet (seperti nomor DANA atau OVO) semata-mata digunakan untuk tujuan pengiriman saldo pencairan Anda.

2. Penggunaan Informasi

Data yang kami kumpulkan hanya digunakan untuk keperluan internal operasional aplikasi, yaitu:

  • Memverifikasi identitas pengguna dan mencegah aktivitas curang (Anti-Fraud).
  • Memproses dan mengirimkan penarikan koin ke e-Wallet Anda.
  • Memberikan dukungan layanan pelanggan (Customer Service).

3. Keamanan & Berbagi Data

Kami TIDAK PERNAH menjual, menyewakan, atau menukar data pribadi Anda kepada pihak ketiga mana pun. Data Anda disimpan dengan aman menggunakan sistem enkripsi standar industri pada server cloud kami (Google Cloud/Firebase).

4. Permintaan Penghapusan Data

Jika Anda ingin berhenti menggunakan aplikasi kami dan ingin seluruh data Anda dihapus dari server kami, Anda dapat menghubungi tim dukungan kami melalui WhatsApp atau Email resmi yang tertera di halaman utama website.
